# Documentation

- `architecture.md` follows a command from input to process exit.
- `abi.md` documents the executable formats and WebAssembly imports.
- `syscalls.md` records the userland capability surface.
- `porting.md` is the shortest route to adding a program or host.

The root `README.md` remains the run book and source of truth for build and
launch commands.
